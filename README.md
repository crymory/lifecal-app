# Goal Dots Wallpaper Generator

Next.js приложение для генерации живых обоев iPhone, которые отслеживают прогресс ваших целей. Генерирует PNG с сеткой затухающих точек, когда дни проходят.

## Локальный запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Как использовать

1. **Настроить параметры** на левой панели (название цели, цвет, даты, модель iPhone).
2. **Скопировать ссылку** (кнопка "Копировать") для автоматизации на iPhone.
3. **В приложении Shortcuts** на iPhone:
   - Создать новую автоматизацию (по времени, ежедневно)
   - Добавить действие: **Получить содержимое URL** → вставить скопированную ссылку
   - Добавить действие: **Установить фото обоев** → выбрать экран блокировки
4. **Изображение будет обновляться автоматически** каждый день.

## URL параметры

Ссылка строится с параметрами:
- `goal=...` — название цели
- `start_date=YYYY-MM-DD` — дата начала
- `goal_date=YYYY-MM-DD` — дата завершения
- `color=...` — HEX цвет
- `res=...` — разрешение (например, 1170x2532)
- `direct=true` — сразу скачать изображение
