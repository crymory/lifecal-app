# Goal Dots Wallpaper Generator

Next.js приложение для генерации живых обоев iPhone, которые отслеживают прогресс ваших целей. Генерирует PNG с сеткой затухающих точек, когда дни проходят.

## Локальный запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Как использовать

1. **На главной странице** заполните параметры обойны:
   - Название цели (Goal)
   - Акцентный цвет (Color)
   - Модель iPhone
   - Даты начала и окончания

2. **Скопируйте ссылку** кнопкой "Copy" — эта ссылка будет использоваться в Shortcuts

3. **На iPhone** в приложении **Shortcuts**:
   - Перейдите на вкладку **Automation** → **New Automation**
   - Выберите **Time of Day** (например, 6:00 AM)
   - Повтор: **Daily**, запуск: **Run Immediately**
   - Нажмите **Create New Shortcut**

4. **В автоматизации** добавьте действия:
   - **Get Contents of URL** → вставьте скопированную ссылку
   - **Set Wallpaper Photo** → **Lock Screen**
   - В опциях **Set Wallpaper Photo** отключите "Crop to Subject" и "Show Preview"

5. **Готово!** Обойна будет обновляться ежедневно автоматически

## Как работает

- При открытии ссылки (например, в Shortcuts или браузере) происходит:
  1. Отрисовка canvas с вашими параметрами (цель, даты, цвет)
  2. Автоматическое скачивание PNG на устройство
  3. В iPhone это работает в Shortcuts прямо во время автоматизации

## URL параметры

Ссылка строится с параметрами:
- `goal=...` — название цели
- `start_date=YYYY-MM-DD` — дата начала
- `goal_date=YYYY-MM-DD` — дата завершения
- `color=...` — HEX цвет
- `res=...` — разрешение (например, 1170x2532)

Пример:
```
https://yourapp.com/wallpaper?goal=Marathon&start_date=2026-02-21&goal_date=2026-04-21&color=%23FF6B4A&res=1170x2532
```
